Learning

This journey of learning the skills to be a competent, junior full-stack software developer has not only taught me the necessary skills to start this new career, but it has also taught me how to learn.
I thought I had a decent grasp of my how I can improve my knowledge. I would watch tutorial videos and rewatch them while following along. I would read through articles and take notes as I went through. This felt like it was enough. I felt like my skillset improved steadily. That was until I tackled some of the back-end learning material at Code Institute.
The Python and Django sections involved reading through theory and watching short explainer videos. We would then be asked some questions where we were encouraged to answer. Thorough explanations would then be offered before being presented with a coding challenge. If we were stuck, we could check hints. After this we would have access to the solution. The constant, bite-sized mix of learning material, questions, and challenges helped to cement a lot of knowledge, quickly.
It has been an empowering and insightful journey. I will utilise this method of learning continue my development.
